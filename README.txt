קניות בסופר משפחת שרון - גרסה 4.6

שינויים:
1. במאגר המוצרים נוספו כפתורי ✏️ עריכה ו-🗑️ מחיקה.
2. עריכת מוצר מאפשרת שינוי שם וקטגוריה.
3. מחיקת מוצר מסירה אותו מהרשימה אם הוא כבר נוסף לקנייה.
4. כאשר נכנסים לקטגוריה ספציפית, רשימת הקניות מוסתרת ומוצג רק המאגר של אותה קטגוריה.
5. הסנכרון נשמר דרך Firebase.
