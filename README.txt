קניות בסופר משפחת שרון - גרסה 4.1 Cloud Sync

מה חדש:
1. חיבור ל-Firebase Realtime Database.
2. סנכרון רשימת הקניות בין כמה טלפונים.
3. מוצרים חדשים נשמרים במאגר המשותף בענן.
4. סימון "נקנה" מוחק את הפריט אצל כולם.

חשוב:
ב-Firebase יש לוודא שב-Realtime Database > Rules יש הרשאות קריאה/כתיבה מתאימות.
לבדיקה ראשונית בלבד אפשר:
{
  "rules": {
    ".read": true,
    ".write": true
  }
}
לאחר שהכול עובד, מומלץ להקשיח הרשאות.
